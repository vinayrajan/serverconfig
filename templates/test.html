Get All Products end point 

http://localhost:5000/products/json
Success - returns the json of all the products added by you
Failure - json with no products

Get Single Product end point 

http://localhost:5000/product/<Product-Id>/json
Example - http://localhost:5000/product/1/json
Success - returns the json of the product
Failure - json with no products

Get All Categories end point 

http://localhost:5000/categories/json
Success - returns the json of all the categories added by you
Failure - json with no categories

Get Single Category end point 

http://localhost:5000/category/<Category-Id>/json
Example - http://localhost:5000/category/1/json
Success - returns the json of the category
Failure - json with no products
